Atividade Prática
Página Web com Tabela, Link e Imagem
Desenvolver duas páginas HTML interligadas, utilizando corretamente:
Estrutura básica do HTML
Tabela com mesclagem de linhas e colunas
Inserção de imagens
Links internos e externos
Organização adequada do código
 
Descrição da Atividade
 
Você deverá criar um pequeno site composto por duas páginas HTML:
index.html
pagina2.html
O tema do site é livre (exemplos: cursos, filmes, jogos, livros, tecnologia, esportes, viagens, séries, etc.), porém a estrutura deve seguir as exigências abaixo.
 
 
Página 1 – index.html
 
Esta será a página principal do site.
 
Deve conter:
Estrutura básica completa do HTML:
<!DOCTYPE html>
<html>
<head>
<body>
Um:
Título principal (<h1>)
Parágrafo explicativo sobre o tema escolhido
Uma tabela contendo:
Pelo menos 4 itens relacionados ao tema
Cabeçalho com colspan
Uso de rowspan em pelo menos uma coluna
Mínimo de 4 colunas (exemplo: categoria, nome, descrição, imagem, link)
A tabela deve conter:
Pelo menos uma imagem por item
Pelo menos:
1 link interno (para pagina2.html)
1 link externo (abrindo em nova aba com target="_blank")
Uma linha final da tabela com:
Total de itens
Uso de colspan
 
 
Página 2 – pagina2.html
 
Esta página deverá conter informações detalhadas sobre um dos itens apresentados na tabela.
 
Deve conter:
Título (<h1>)
Pelo menos 2 parágrafos explicativos
Uma lista (<ul> ou <ol>)
Uma imagem relacionada ao conteúdo
Um link para voltar para index.html
 
Organização dos Arquivos
Os dois arquivos devem estar na mesma pasta, permitindo a navegação correta entre eles.
